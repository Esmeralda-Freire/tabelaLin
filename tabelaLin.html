Linguagem Características da linguagem Comentário pessoal
Javascript Sintaxe familiar, controle sobre HTML e CSS, independente de plataforma,
linguagem interpretada e possui modelo de objetos baseado em protótipos.		 Fácil de entender e bastante didática.
Java Orientada a objetos, segura, portável, independente de plataforma,
interpretada e compilada.		 Pelo pouco contato que tive com a linguagem achei complicada para iniciantes.
PHP Simplicidade, multiplataforma, código aberto, velocidade e confiabilidade. Ainda não tive nenhum contato com o PHP.
Python Grande comunidade, multiplataforma, curva de aprendizado baixa,
tipagem forte e dinâmica.		 Muito dinâmica para programadores iniciantes, sintaxe simples.
C# Fortemente tipada, orientada a objetos e componentes, case-sensitive,
rápida e versátil.		 Também ainda não tive nenhum contato com o C#.
C Modularidade, fácil de entender, portabilidade, fácil e eficiente. Não mais dinâmica que o Python, porém de fácil entendimento.
